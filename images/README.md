# 🃏 مافيا | العالم السفلي

لعبة مافيا أونلاين تُلعب من المتصفح مباشرة — كل لاعب من جهازه الخاص، والتزامن بين اللاعبين يتم عبر **Firebase Realtime Database** بدون الحاجة لسيرفر خلفي دائم.

## ✨ المميزات
- استضافة بوضعين: **شاشة بث** (المضيف يشاهد فقط) أو **المضيف يلعب مع الجميع**.
- انضمام برمز غرفة من 6 أرقام أو عبر QR Code.
- أدوار: 🔍 محقق، 🩺 طبيب، 🔪 مافيا، 🥷 حرامي (يظهر تلقائيًا إذا زاد عدد اللاعبين عن 9)، 👤 مواطن.
- المافيا والحرامي يتعرفون على بعض تلقائيًا ولديهم شات خاص لتنسيق هدف الليل.
- بطاقات كشف الأدوار بصور مخصصة (انظر مجلد `images/`) مع أنيميشن دخول دوّار وتصغير تلقائي لبطاقة دائمة.
- مراحل: كشف الأدوار → الليل (طبيب/محقق/مافيا) → النهار (نقاش) → التصويت.
- أزرار تخطي للمضيف: تخطي جولة النقاش بالكامل، أو تخطي التصويت بدون إقصاء أحد، في حال عدم وجود اشتباه.
- شاشة بث احترافية للمضيف المُشاهد تعرض كل مجريات المباراة دون كشف أي دور.
- سجل أحداث جانبي لكل ما يحصل في اللعبة.

## 📁 الملفات
| الملف | الوصف |
|---|---|
| `index.html` | هيكل الصفحة وكل شاشات اللعبة |
| `style.css` | التصميم الكامل (خلفية متحركة، بطاقات، أنيميشن) |
| `script.js` | منطق اللعبة بالكامل + الاتصال بـ Firebase |
| `images/` | صور بطاقات الأدوار (راجع `images/README.txt` للأسماء المطلوبة) |

> ملاحظة: يوجد في المشروع أيضًا `server.js` و`package.json` من نسخة قديمة تجريبية تعتمد على Node.js + Socket.io بدل Firebase. النسخة الفعّالة والمستخدمة حاليًا هي `script.js` (Firebase). لا حاجة لتشغيل `server.js` لتشغيل اللعبة.

## 🚀 التشغيل
اللعبة صفحة استاتيكية بالكامل، ما تحتاج سيرفر:

1. أنشئ مشروع [Firebase](https://console.firebase.google.com) مجاني وفعّل **Realtime Database**.
2. انسخ إعدادات المشروع (`firebaseConfig`) والصقها في أول `script.js` بدل القيم الحالية.
3. افتح `index.html` مباشرة في المتصفح، أو ارفع الملفات على أي استضافة استاتيكية مثل **GitHub Pages** أو **Vercel**.

### النشر على GitHub Pages
1. ادفع الملفات (`index.html`, `style.css`, `script.js`, `images/`) لريبو GitHub.
2. من إعدادات الريبو → **Pages** → اختر الفرع (`main`) والمجلد (`root`) → Save.
3. الرابط بيصير متاح خلال دقائق على `https://username.github.io/repo-name/`.

## 🖼️ صور بطاقات الأدوار
ضع صورك داخل مجلد `images/` بنفس الأسماء الموضحة في `images/README.txt`:
```
role-citizen.png     -> المواطن
role-detective.png   -> المحقق
role-doctor.png      -> الطبيب
role-mafia.png        -> المافيا
role-thief.png        -> الحرامي
```
إذا كانت أي صورة غير موجودة، البطاقة تعمل عادي بدون صورة (بدون أخطاء).

## 🎮 طريقة اللعب
اضغط زر **"شرح اللعبة"** داخل التطبيق نفسه — فيه شرح كامل للأدوار ومراحل اللعبة.
